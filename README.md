# vital-signs

Open source project to analyze signs of software ecosystems' death.

## Redis

Require Redis connection
See more here https://developer.redis.com/howtos/quick-start
After installed run `redis-server` command

## .env config vars

SO_CLIENT_ID
SO_CLIENT_SECRET
SO_CLIENT_KEY
SO_SITE
SO_API_URL
DATABASE_URL
REDIS_HOST
REDIS_PORT
CONCURRENCY
